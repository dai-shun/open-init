## 结构划分

 推荐的目录基本结构为：

```bash
├── /dist/   .         # 编译后代码目录 
├── /src/              # 项目源码目录
│ ├── /components/     # 项目组件
│ ├── /routes/         # 路由组件
│ ├── /actions/        # redux actions  
│ ├── /reducers/       # redux reducers
│ ├── /services/       # 业务逻辑
│ ├── /utils/          # 工具函数
├── package.json       # 项目信息
└── webpack.config.js  # webpack配置
```
