# open-init

[![NPM version](https://img.shields.io/npm/v/open-init.svg?style=flat)](https://npmjs.org/package/open-init)

 快速搭建React开发环境

----

## Feature

- 一行命令生成基于webpack的开发环境
- 支持ES2015

## Install

```bash
$ npm i open-init -g
```

## Usage
搭建一个普通的React+Redux开发环境

```bash
$ mkdir React-Demo && cd React-Demo
$ open-init -n
```

Start development server.

```bash
$ npm start
```

Build.

```bash
$ npm run build
```
